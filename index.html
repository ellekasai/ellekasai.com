Hi, I'm Elle Kasai, a UI/UX Designer in Vancouver.

My Story

I was born in Phillippines and raised in Kanagawa, Japan. After graduating from Gakushuin Women's College with a Marketing degree in 2012, I started working at a payment startup in Tokyo as a sales associate.

But I've always loved to express myself and be creative, and I wasn't quite satisfied with my job. So after a year, I decided to pursue a completely different career in UI design. I left my job, stayed with my boyfriend in Silicon Valley, and learned coding and design from him.

Eventually I came back to Japan, and in Fall 2013, I became a junior designer at BizReach, a tech startup in Tokyo. I've worked on various projects, including coding the entire frontend of BizReach Woman. During my free time, I've taught an online HTML/CSS course and spoke at coding-related events.

When I'm not in front of my computer, I enjoy reading, watching movies, and spending time with my dogs. I was also a model at Ashinari, a Japanese stock photo site. You can view my model photos here.

Talk 2 / Introduction to HTML & CSS Sketching

March - April 2014 / Online, in Japanese

I gave a four-part lecture series on Schoo (a Udemy for Japan) about HTML & CSS Sketching.

Often the fastest way to master HTML and CSS is to replicate existing sites using HTML and CSS. I called this HTML & CSS Sketching and gave live-coding talks on how to sketch a site effectively.

My audience was mostly beginning web designers with some knowledge of HTML and CSS. Each lecture had ~500 live viewers on average.

Each lecture was 90 minutes long. On the first lecture, I sketched a Wikipedia page. On the remaining three lectures, I sketched a user page and a pull request page on GitHub.

Unfortunately, you have to pay to view my talks on Schoo, and both the site and the talks are in Japanese. However, my code is available on GitHub at: htmlsketch/sample.

On lecture #1, I live-coded a Wikipedia page in 90 minutes.

Wikipedia has a simple grid, which allows beginners focus on learning core HTML elements and CSS properties. I also demonstrated the Emmet plugin on Sublime Text.

The resulting code is open-sourced here.

On lecture #2, I live-coded a user page on GitHub in 90 minutes. On lectures #3 and #4, I live-coded a pull request page on GitHub in 180 minutes.

I used xScope to ensure that the colors and measures are accurate. I also used Font Awesome as substitute icons.

The resulting code is open-sourced here and here.
